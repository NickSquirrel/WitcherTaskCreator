# 《五骨圣女》网状任务剧情设计

## 基本定位

**类型：** 巫师风格网状支线任务组  
**主角：** 杰洛特  
**结构：** 五个小型任务 + 一个五幕长线任务  
**地区：** 威伦荒村、诺维格瑞边缘、史凯利格偏远岛屿  
**核心题眼：** 被称为“圣女”的人，究竟是奇迹的源头，还是被人反复利用的牺牲者。  

五个短线任务表面上互不相干：稻草人异动、磨坊食尸生物、修道院圣物失窃、贫民区慈善诅咒、海雾钟声。玩家完成其中任意两个后，会发现这些事件都与“圣梅洛娃”的遗骨有关；完成三个后，会开始怀疑圣女传说被教团、贵族和地方势力改写；完成全部五个后，能拼出梅洛娃从民间医者、逃亡者、牺牲者到被神化工具的完整轨迹。

正史角色以客串方式提供不同类型的证据：丹德里恩提供民谣与口述历史，特莉丝提供草药师和猎巫语境，叶奈法提供高阶咒术判断，希里只在长线末段短暂触碰残留记忆。最终判断仍由杰洛特和玩家承担。

## 梅洛娃的真实经历

### 公开传说

圣梅洛娃被传为“白手圣女”：她在饥荒中让麦田复苏，在瘟疫中让孩子退烧，在海难中平息潮声。教团宣称她一生服从神意，死后遗骨分散各地，继续守护虔诚之人。

### 真实轨迹

梅洛娃原本是威伦一带的民间医者，擅长草药、接生和疫病护理。战争和饥荒后，她发现普通药物救不了所有人，于是接触旧式术士残卷，创造了一种“转嫁病痛”的禁忌仪式。这个仪式能让某个聚落暂时免于瘟疫、歉收或海难，但灾厄不会消失，只会被转移给被排除在共同体之外的人：流民、无人认领的死者、非人种族、敌对氏族或穷人。

她最初以为自己是在争取时间，后来发现贵族和教团把她的能力制度化：让“奇迹”保护有用的人，让没有名姓的人承担代价。晚年她逃往史凯利格，试图把仪式封死。她临死前要求把自己的遗骨分开，因为完整圣骨会恢复“选择牺牲者”的权柄。

### 揭示节奏

| 阶段 | 玩家认知 | 代表线索 |
|---|---|---|
| 任意 1 个短线 | 圣物确实有效，但保护方式异常。 | 稻草人、磨坊水渠、梦游救济者或海雾钟声。 |
| 任意 2 个短线 | 圣物不是单纯祝福，效果依赖转嫁。 | 病痛、歉收、海难或诅咒从一群人转移到另一群人。 |
| 任意 3 个短线 | “圣女”身份被后人包装。 | 被篡改民谣、修道院审讯记录、草药师残页。 |
| 任意 4 个短线 | 梅洛娃曾主动反抗教团与贵族。 | 被禁歌谣、非人口述史、史凯利格旧誓。 |
| 全部 5 个短线 | 她临死前要求分尸封印自己。 | 海底咒式、希里的记忆感应、五骨归龛仪式图。 |

## 共享变量

| 变量 | 默认值 | 写入来源 | 读取效果 |
|---|---:|---|---|
| `melowa_truth` | 0 | 所有短线任务 | 决定长线任务中可用推理、最终对白和结局选项。 |
| `public_truth` | 0 | 《磨坊下的白水》《红蜡印》 | 决定民众能否接受圣女真相，影响公开真相结局。 |
| `witch_hunter_attention` | 0 | 《红蜡印》《没有脸的施主》 | 决定诺维格瑞猎巫压力，影响特莉丝、草药师和非人 NPC 安全。 |
| `blackmail_evidence` | false | 《没有脸的施主》 | 决定最终能否压制教团并阻止他们抢夺圣骨。 |
| `clan_oath` | 0 | 《潮下钟声》 | 决定史凯利格氏族在最终任务中护送、旁观或追杀杰洛特。 |
| `village_fear` | 0 | 《稻草人的祷词》《磨坊下的白水》 | 决定威伦村民是否协助公开旧事，或继续依附圣物谎言。 |

`melowa_truth` 建议用 0 到 5 计数，每个短线任务完成并取得核心证据时 +1。其他变量可用低、中、高或布尔值实现。

## 短线任务 1：《稻草人的祷词》

**地点：** 威伦，荒村“灰垄”  
**类型：** 猎魔委托 / 民俗诅咒  
**表面问题：** 麦田里的稻草人夜间行走，农夫失踪，牲畜被挂在田边。  
**圣骨：** 梅洛娃的一截指骨，被缝在稻草人胸口。  
**客串：** 丹德里恩不现身，但留下早年采风歌稿。  

### 主要角色

- **村长奥尔班：** 守着歉收村庄的老村长，知道圣骨秘密，但相信这是唯一活路。
- **寡妇米拉：** 失踪农夫的妻子，怀疑丈夫不是被怪物带走，而是被村人选中。
- **哑童托马：** 常在麦田边玩，听过稻草人夜里说话。
- **流浪说书人波丹：** 藏有丹德里恩早年歌稿的抄本，用它换酒钱。

### 调查流程

1. 告示板委托称“麦田里有妖灵”，村长要求杰洛特在收割日前解决。
2. 杰洛特检查麦田，发现稻草人没有普通脚印，而是被根须拖行；失踪者血迹只到田埂为止。
3. 米拉透露每年歉收时都会有人“离村”，但村长说他们只是去找活路。
4. 波丹唱出丹德里恩旧歌原词：“白手姑娘给麦穗包扎”，与村民祷词“白手圣女拿走病根”明显不同。
5. 托马指向稻草人体内的指骨，杰洛特可用感官发现骨上有旧式转嫁咒。

### 梅洛娃线索

玩家得知梅洛娃年轻时曾在威伦救治饥民，因常用白布包扎病人的手，被称为“麦田里的白手”。原始传说赞颂她的医术，后来的村社祷词则把她变成会“拿走多余之人”的圣女。

### 选择点

- **烧毁指骨：** 稻草人停止活动，村庄失去“保收”效果，村民短期陷入恐慌。
- **把指骨交还村长：** 委托快速完成，村庄继续丰收，但未来还会有人被选中。
- **把指骨交给波丹带去诺维格瑞：** 村庄失去圣物，丹德里恩线索提前开启，但村民会指责杰洛特破坏活路。

### 状态影响

- 烧毁指骨：`village_fear +1`，`melowa_truth +1`
- 交还村长：`village_fear -1`，`melowa_truth +0`
- 交给波丹：`village_fear +1`，`melowa_truth +1`，提前解锁丹德里恩相关线索

### 后续回响

如果 `village_fear` 高，长线后期威伦村民会拒绝作证，甚至把杰洛特称为“断粮的白狼”。如果玩家保留旧歌证据，丹德里恩能在第四幕证明圣女传说被后人篡改。

## 短线任务 2：《磨坊下的白水》

**地点：** 威伦旧磨坊“白水轮”  
**类型：** 食尸生物调查 / 家庭道德困境  
**表面问题：** 水渠里出现食尸生物，夜间传出婴儿哭声。  
**圣骨：** 梅洛娃的一段肋骨，被藏在磨坊水轮轴心。  
**客串：** 特莉丝通过信件提供药方判断。  

### 主要角色

- **磨坊主卡斯帕：** 用圣骨换来孩子健康，拒绝承认代价。
- **莉达：** 卡斯帕的妻子，知道更多，但相信失去圣骨孩子会死。
- **草药师伊蕾娜：** 曾抄录梅洛娃药方，被村人称为女巫。
- **病童亚涅克：** 被圣骨保护的孩子，同时也是诅咒的锚点。

### 调查流程

1. 杰洛特查看水渠尸骸，发现食尸生物不吃新鲜尸体，只围着水轮徘徊。
2. 伊蕾娜承认自己见过类似药方，但药方边缘有术士符号，她不敢使用。
3. 玩家可选择把符号拓印寄给特莉丝。稍后收到回信：符号不是治愈，而是“病痛转移标记”。
4. 水渠深处发现无人认领者的骨灰和小布鞋，说明病痛被转给了死者与贫民孩子。
5. 卡斯帕承认他知道磨坊“会哭”，但认为一个活着的孩子比无名死者重要。

### 梅洛娃线索

玩家得知梅洛娃曾用禁忌药方救过瘟疫中的孩子。她早期可能相信这只是“借死者余温压住病”，尚未意识到活人的病会被系统性转嫁给弱者。

### 选择点

- **公开真相：** 村民得知孩子健康建立在他人代价上，卡斯帕一家被驱逐或处死。
- **私下解除诅咒：** 亚涅克可能病情复发，但不会引发村庄暴力。
- **让夫妇带孩子逃走：** 保住一家人，圣骨失踪，村庄继续怀疑草药师。

### 状态影响

- 公开真相：`public_truth +1`，`village_fear +1`，`melowa_truth +1`
- 私下解除：`public_truth +0`，`village_fear -1`，`melowa_truth +1`
- 放走夫妇：`public_truth -1`，`melowa_truth +1`

### 后续回响

如果公开真相且 `public_truth` 后续继续提高，长线最终可说服一部分民众接受梅洛娃并非圣女。如果选择隐瞒，最终公开真相时更容易引发暴民清算。

## 短线任务 3：《红蜡印》

**地点：** 诺维格瑞城外，圣卡莱门修道院  
**类型：** 圣物失窃 / 猎巫政治  
**表面问题：** 修道院收藏的圣物失窃，院长雇杰洛特追查。  
**圣骨：** 梅洛娃的一枚掌骨，被封在红蜡圣匣中。  
**客串：** 特莉丝短暂现身，帮助阅读烧毁残页。  

### 主要角色

- **院长维克托：** 宣称圣物被亵渎，实际参与倒卖。
- **猎巫人诺瓦克：** 想用圣骨制造“神迹证据”，扩大清洗。
- **草药师芙蕾娜：** 被栽赃为盗贼，因为她祖母曾保存梅洛娃药方。
- **特莉丝：** 冒险短暂现身，只协助解读魔法残页，不愿扩大冲突。

### 调查流程

1. 修道院称圣匣被夜间闯入者偷走，但门锁没有破坏痕迹。
2. 杰洛特发现红蜡印不是被撬开，而是由院长本人的戒指开启。
3. 地窖中有被烧毁的审讯记录，特莉丝可帮助恢复残页文字。
4. 残页显示梅洛娃曾被审讯，罪名是“未经神殿许可行奇迹”，而非传统巫术。
5. 猎巫人诺瓦克准备公开处决芙蕾娜，把圣骨失窃归咎于草药师。

### 梅洛娃线索

玩家得知教团早就知道梅洛娃的能力，并试图把她纳入神殿权威。她拒绝后，被塑造成“死后才归顺神意”的圣女。

### 选择点

- **揭露院长：** 修道院名誉受损，猎巫人注意力上升。
- **救走芙蕾娜并销毁证据：** 保住无辜者，但教团继续拥有话语权。
- **把证据交给地下情报商：** 获得政治筹码，但证据会被用来勒索而非公开真相。

### 状态影响

- 揭露院长：`public_truth +1`，`witch_hunter_attention +1`，`melowa_truth +1`
- 救走芙蕾娜：`witch_hunter_attention -1`，`melowa_truth +1`
- 交给情报商：`blackmail_evidence = true`，`melowa_truth +1`

### 后续回响

如果 `witch_hunter_attention` 高，诺维格瑞后续会出现更多搜捕，特莉丝相关场景更危险。如果保留审讯残页，长线第二幕可提前确认梅洛娃被制度化利用。

## 短线任务 4：《没有脸的施主》

**地点：** 诺维格瑞贫民区“盐沟”  
**类型：** 诅咒慈善 / 非人保护 / 变形怪调查  
**表面问题：** 匿名施主救济穷人，但接受救济者陆续梦游自残。  
**圣骨：** 梅洛娃的另一枚掌骨，被嵌在施粥锅底的银环里。  
**客串：** 丹德里恩现身，认出一段被禁民谣。  

### 主要角色

- **“没有脸的施主”艾勒：** 变形怪，伪装成人类慈善家。
- **修女玛塔：** 真心救济穷人，但接受教团压力。
- **矮人铁匠布洛克：** 失去亲人，怀疑施粥点有问题。
- **丹德里恩：** 在酒馆认出旧民谣，帮助找见证人后代。

### 调查流程

1. 杰洛特调查梦游者，发现他们并非被控制，而是在梦里反复承受别人的病痛记忆。
2. 施粥锅底有银环，银环中嵌着掌骨，能让受救济者短期免病。
3. 丹德里恩听见贫民哼唱的禁歌，认出其中一段不是圣歌，而是逃亡者挽歌。
4. 见证人后代透露：梅洛娃曾保护非人和疫病孤儿，后来被人类权贵交给教团。
5. 艾勒承认他用圣骨维持救济点，因为教团承诺只要“奇迹”继续，就暂不清洗贫民区。

### 梅洛娃线索

玩家得知梅洛娃不是只救“虔诚的人”。她曾把非人、弃婴和病人藏在城市边缘，因此得罪贵族与教团。圣女传说删掉了这部分经历。

### 选择点

- **揭露艾勒：** 变形怪身份曝光，教团借机扩大清洗。
- **保护艾勒并毁掉圣骨：** 救济点失去奇迹，贫民短期受苦，但清洗借口变弱。
- **逼艾勒交出教团黑账：** 获得政治证据，但艾勒和救济点可能被报复。

### 状态影响

- 揭露艾勒：`witch_hunter_attention +1`，`melowa_truth +1`
- 保护艾勒：`witch_hunter_attention -1`，`public_truth +1`，`melowa_truth +1`
- 取得黑账：`blackmail_evidence = true`，`witch_hunter_attention +1`，`melowa_truth +1`

### 后续回响

如果 `blackmail_evidence` 为 true，长线最终可用教团黑账阻止他们公开夺取圣骨。如果 `witch_hunter_attention` 过高，艾勒或芙蕾娜可能在最终前死亡或失踪。

## 短线任务 5：《潮下钟声》

**地点：** 史凯利格偏远岛屿“洛恩维克”  
**类型：** 海岛诅咒 / 氏族誓约 / 水下遗骨  
**表面问题：** 海雾中有钟声引船触礁，岛民认为是海妖作祟。  
**圣骨：** 梅洛娃的一截腿骨，被沉在海底旧钟内。  
**客串：** 叶奈法现身判断咒式性质。  

### 主要角色

- **氏族长哈尔格：** 维护旧誓，认为外人不该碰海底圣骨。
- **寡妇凯拉赫：** 丈夫死于触礁，要求知道真相。
- **走私者维德：** 试图打捞圣骨卖给诺维格瑞买家。
- **叶奈法：** 判断海底咒式不是封印怪物，而是封印选择牺牲者的权柄。

### 调查流程

1. 杰洛特检查触礁船只，发现船员不是被海妖诱惑，而是听见熟人呼救声。
2. 岛上旧钟楼没有钟，海底却传来同频钟声。
3. 叶奈法读取海底咒式，指出它将完整仪式的“牺牲者选择权”锁在海中。
4. 氏族长承认祖辈收留过晚年的梅洛娃，并按她遗愿把腿骨沉海。
5. 走私者试图趁杰洛特调查时打捞圣骨，引发雾中幻声失控。

### 梅洛娃线索

玩家得知梅洛娃晚年已经意识到自己的遗骨会被利用。她逃到史凯利格，请当地氏族把五骨分散，并把最关键的一块沉入海底，防止任何人恢复完整仪式。

### 选择点

- **维持氏族秘密：** 岛上秩序稳定，死者家属得不到真相。
- **公开梅洛娃遗愿：** 氏族名誉受损，但最终可获得道义支持。
- **带走腿骨：** 完成五骨归龛条件，但氏族可能追杀杰洛特。

### 状态影响

- 维持秘密：`clan_oath +0`，`melowa_truth +1`
- 公开遗愿：`clan_oath +1`，`public_truth +1`，`melowa_truth +1`
- 带走腿骨：`clan_oath -1`，`melowa_truth +1`

### 后续回响

如果 `clan_oath` 高，最终任务中史凯利格人会护送杰洛特运送或销毁圣骨。如果为负，氏族会把杰洛特视为盗骨者。

## 长线任务：《五骨归龛》

**触发方式：** 完成任意两个短线任务后自动出现第一幕；完成任意三个短线任务后进入第二幕；第五幕需要至少四个核心证据或全部五个短线任务。  
**核心目标：** 决定梅洛娃遗骨和她真实故事的归宿。  

### 第一幕：圣女传说

杰洛特发现多个地区都在供奉同一位“圣梅洛娃”，但祷词互相矛盾。威伦称她让麦田复苏，诺维格瑞称她服从神殿，史凯利格称她要求永远沉默。

**读取变量：** `melowa_truth >= 2`  
**关键场景：** 杰洛特整理圣骨线索，发现每块圣骨都保护一群人，同时伤害另一群人。  
**推进结果：** 长线任务正式命名为《五骨归龛》。

### 第二幕：医者真相

特莉丝确认梅洛娃不属于学院或神殿，而是民间医者。她掌握的禁忌药方可能来自战乱中流散的术士残卷，但她最初并非权力人物。

**读取变量：** `public_truth`、`witch_hunter_attention`  
**关键场景：** 若猎巫压力高，特莉丝只能通过密信提供判断；若压力低，她可短暂现身协助解读残页。  
**推进结果：** 玩家确认梅洛娃的“奇迹”源于医术与禁忌仪式的混合。

### 第三幕：禁忌代价

叶奈法揭示完整圣骨仪式不是治疗，而是转嫁。它可以精准选择谁承担灾厄，因此才会被教团、贵族、村社和氏族争夺。

**读取变量：** `clan_oath`、`melowa_truth`  
**关键场景：** 若史凯利格线处理得当，叶奈法能在海底钟咒中确认梅洛娃封印的核心；若没有，她只能从残缺咒式推断，最终选项减少。  
**推进结果：** 玩家知道“重组圣骨”不是收集神器，而是恢复一种选择牺牲者的权力。

### 第四幕：被制造的圣女

丹德里恩找到旧歌源头，证明最初的歌谣讲的是医者、逃亡者和孤儿，而不是服从神殿的圣女。教团和贵族共同改写了她的故事，把她死后的遗骨变成统治工具。

**读取变量：** `blackmail_evidence`、`public_truth`、`village_fear`  
**关键场景：** 若玩家拥有黑账，丹德里恩能帮助把证据传播出去；若没有，公开真相会被教团反咬为异端谣言。  
**推进结果：** 最终前，玩家可选择倾向保密、公开、勒索或摧毁。

### 第五幕：最后见证

希里短暂出现。她不参与调查，也不替杰洛特选择，只是在完整或接近完整的圣骨旁感应到残留记忆：梅洛娃临死前并非被动成为圣女，而是主动要求分尸封印自己，防止后人掌握转嫁权柄。

**读取变量：** `melowa_truth >= 4`  
**关键场景：** 若 `melowa_truth` 不足，希里只能看到破碎片段，玩家无法完全确认梅洛娃遗愿。  
**推进结果：** 开启最终选择。

## 最终选择与结局矩阵

### 结局 A：重组圣骨，执行仪式

**做法：** 把五骨归龛，选择一个群体承受瘟疫或灾厄。  
**条件：** 至少取得四块圣骨或等价仪式信息。  
**结果：** 大多数城镇得救，教团或地方权力会把这包装成新奇迹。杰洛特得到报酬，但梅洛娃最害怕的事情发生。  
**代价：** 指定群体死亡或流亡，后续世界更稳定也更冷酷。

### 结局 B：摧毁全部遗骨

**做法：** 用叶奈法提供的反咒和猎魔手段摧毁遗骨锚点。  
**条件：** `melowa_truth >= 4`，并掌握关键咒式。  
**结果：** 转嫁体系终结，但被压住的瘟疫和灾厄局部爆发。  
**代价：** 短期死伤更高，杰洛特被很多人怨恨；梅洛娃终于不再被使用。

### 结局 C：公开真相并分散遗骨

**做法：** 公开梅洛娃真实经历，同时让各地分别封存或监管遗骨。  
**条件：** `public_truth` 较高，并有丹德里恩歌谣证据或教团黑账。  
**结果：** 教团权威受损，民众开始争夺叙事权。部分地区建立新的防疫和救济制度。  
**代价：** 社会动荡、暴民复仇、贵族清算和信仰崩塌不可避免。

### 结局 D：交给某一方

**做法：** 把圣骨或关键证据交给教团、史凯利格氏族、地下情报商或地方联盟。  
**条件：** 对应派系仍存活且愿意交易。  
**结果：** 杰洛特得到明确报酬和局部稳定。  
**代价：** 梅洛娃再次被改写成工具。不同派系会创造不同版本的新圣女传说。

## 正史角色使用原则

- **叶奈法：** 只负责高阶咒术判断。她会指出仪式代价，但不会替杰洛特决定是否摧毁或使用圣骨。
- **特莉丝：** 关注草药师、非人和猎巫风险。她的态度会提醒玩家：公开真相可能让无辜者先死。
- **丹德里恩：** 负责民谣、传说篡改和口述历史。他让玩家看到“历史如何被唱错”。
- **希里：** 只在末段短暂登场，用特殊感应补足梅洛娃临终真相。她不参与战斗、不接管主线。

## 验证标准

- 完成任意两个短线任务后，玩家应意识到圣物不是纯粹祝福。
- 完成任意三个短线任务后，玩家应怀疑“圣女”称号是后人包装。
- 完成全部五个短线任务后，玩家能拼出梅洛娃完整人生轨迹。
- 每个短线任务都有独立道德困境，同时至少写入一个长线变量。
- 正史角色提供证据维度，而不是替代杰洛特调查。
- 最佳结局仍保留至少一种代价：信仰崩塌、瘟疫扩散、政治清算或地区混乱。
