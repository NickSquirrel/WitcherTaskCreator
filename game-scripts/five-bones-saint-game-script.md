# 《五骨圣女》正式游戏剧本

## 文档用途

本剧本用于把《五骨圣女》从剧情设计扩展为可实现的 RPG 任务脚本。内容包含任务包结构、变量接口、场景触发、任务目标、对白、分支选择、状态写入和结局播放条件。对白为正式可用草案，可继续交给关卡、叙事、任务脚本和本地化人员细化。

## 全局设定

**主角：** 杰洛特  
**任务包名称：** 五骨圣女  
**任务数量：** 五个短线任务 + 一个长线汇聚任务  
**主线推荐完成顺序：** 威伦 1 -> 威伦 2 -> 诺维格瑞 1 -> 诺维格瑞 2 -> 史凯利格 -> 五骨归龛  
**允许乱序：** 是。长线任务根据核心证据数量分幕推进。  

## 全局变量

| 变量 | 类型 | 初始值 | 用途 |
|---|---|---:|---|
| `melowa_truth` | int | 0 | 梅洛娃真相拼图。每取得一条核心证据 +1。 |
| `public_truth` | int | 0 | 民众接受真相程度。影响公开结局可用性。 |
| `witch_hunter_attention` | int | 0 | 猎巫压力。影响诺维格瑞安全状态。 |
| `blackmail_evidence` | bool | false | 是否掌握教团黑账。影响压制教团选项。 |
| `clan_oath` | int | 0 | 史凯利格氏族态度。影响最终护送或追杀。 |
| `village_fear` | int | 0 | 威伦村社恐惧程度。影响证人配合。 |
| `bone_finger` | enum | none | 指骨归属：none / geralt / village / bard / destroyed。 |
| `bone_rib` | enum | none | 肋骨归属：none / geralt / buried / family_lost。 |
| `bone_palm_church` | enum | none | 修道院掌骨归属。 |
| `bone_palm_soup` | enum | none | 盐沟掌骨归属。 |
| `bone_leg` | enum | none | 腿骨归属。 |

## 主要角色表

| 角色 | 功能 | 使用原则 |
|---|---|---|
| 杰洛特 | 玩家角色、调查者、最终选择承担者 | 不被正史客串角色替代决策。 |
| 梅洛娃 | 失声的核心人物 | 通过遗骨、歌谣、审讯记录和记忆显现逐步揭示。 |
| 丹德里恩 | 民谣与口述历史线索 | 提供传说被改写的证据。 |
| 特莉丝 | 草药、猎巫和非人安全线索 | 提供符号判断与风险提醒。 |
| 叶奈法 | 高阶咒术线索 | 明确完整圣骨仪式的真实机制。 |
| 希里 | 最终记忆见证 | 只在尾声前补全梅洛娃遗愿。 |

## 任务 1：《稻草人的祷词》

### 任务日志

**初始日志：**  
灰垄村的村民说麦田里的稻草人夜里会走动，还带走了一个农夫。听上去像乡下人的迷信，不过乡下人的迷信常常能走路、吃人，还要价不低。

**真相日志：**  
稻草人体内藏着一截指骨。村民称它属于圣梅洛娃。只要把骨头缝进稻草人，麦子就能活过歉收年。代价是每年有人被“带走”。

### 场景 1.1：告示板与村长

**地点：** 灰垄村口  
**参与者：** 杰洛特、村长奥尔班、围观村民  
**触发：** 玩家读取告示或进入村口。  
**目标更新：** “和灰垄村村长谈谈。”  

**对白：**

奥尔班：杀了它，白狼。烧了、砍了、埋了，都行。只要收割日前别再让它走。  
杰洛特：它？  
奥尔班：稻草人。  
杰洛特：稻草人通常不需要猎魔人。  
奥尔班：这个需要。它夜里会唱，还会挑人。  
杰洛特：三天前开始的？  
奥尔班：你怎么知道？  
杰洛特：你看上去像三天没睡。三天前发生了什么？  
奥尔班：没什么。  
杰洛特：说“没什么”的地方，尸体通常埋得浅。

### 场景 1.2：麦田调查

**地点：** 灰垄麦田  
**玩法：** 猎魔感官调查。  
**线索：**

- 被拖行的麦根：不是脚印，是地下根须牵引。
- 失踪农夫血迹：停在第三个稻草人脚下。
- 白布碎片：绣有白手图案。
- 托马的指向：哑童指向最大稻草人。

**目标更新：** “检查田中央最大的稻草人。”  

### 场景 1.3：稻草人体内的指骨

**地点：** 麦田中央  
**参与者：** 杰洛特、托马、村长奥尔班、米拉  
**触发：** 玩家剖开稻草人。  

**对白：**

杰洛特：里面不是稻草。  
奥尔班：别碰它。  
杰洛特：委托我来碰的就是它。  
米拉：那是我丈夫的血吗？  
杰洛特：是。到这里就没了。  
奥尔班：圣女拿走该拿走的人。  
米拉：谁决定“该”？圣女？还是你？

### 场景 1.4：旧歌稿

**地点：** 酒馆外  
**参与者：** 杰洛特、流浪说书人波丹  
**触发：** 玩家追上波丹或用 Axii 留住他。  

**对白：**

波丹：我只是唱歌的。村里人给酒，我唱他们爱听的。  
杰洛特：这首不是他们写的。  
波丹：当然不是。老稿子，丹德里恩早年写的。可原词没人爱听。  
杰洛特：原词是什么？  
波丹：白手姑娘给麦穗包扎。后来他们改成，白手圣女拿走病根。更像祷词，也更吓人。

### 场景 1.5：选择

**选择 A：烧毁指骨**  
奥尔班：你烧的不是骨头，是我们过冬的粮。  
杰洛特：如果粮要用人命浇，早晚会有人问这是谁的冬天。  
**写入：** `bone_finger = destroyed`，`village_fear += 1`，`melowa_truth += 1`

**选择 B：交还村长**  
米拉：那我丈夫呢？  
杰洛特：我只能告诉你真相。不能替全村人接受它。  
**写入：** `bone_finger = village`，`village_fear -= 1`

**选择 C：交给波丹带去诺维格瑞**  
波丹：你让我带着圣物上路？我看起来像活够了吗？  
杰洛特：你看起来像知道怎么把故事传出去。  
**写入：** `bone_finger = bard`，`village_fear += 1`，`melowa_truth += 1`

## 任务 2：《磨坊下的白水》

### 任务日志

白水轮磨坊水渠里出现了食尸生物。但怪物不像在觅食，更像在守着什么东西。磨坊主的孩子病得太久，也活得太巧。

### 场景 2.1：尸骸与水轮

**地点：** 白水轮磨坊  
**玩法：** 战斗 + 调查。  
**敌人：** 2-3 只低阶食尸生物。  
**特殊行为：** 敌人会回避水轮轴心，不会靠近肋骨位置。  

**杰洛特自语：**  
食尸生物不吃尸体，反倒怕水轮。不是巢，是边界。

### 场景 2.2：草药师伊蕾娜

**地点：** 草药师小屋  
**参与者：** 杰洛特、伊蕾娜  

伊蕾娜：梅洛娃的药方？我有抄本。没有人敢用。  
杰洛特：为什么？  
伊蕾娜：药方能退烧，也能让别的地方发烧。你明白吗？  
杰洛特：明白得不够。  
伊蕾娜：我也一样。所以我没碰它。

**可选动作：** 将符号拓印寄给特莉丝。  
**回信内容：**

特莉丝：这不是治愈符号，是转移标记。若你看到水、骨灰和儿童物件，停止仪式。还有，别让村民把草药师当成答案烧掉。那是他们最擅长的事。

### 场景 2.3：磨坊主的供认

**地点：** 水轮下方  
**参与者：** 杰洛特、卡斯帕、莉达、亚涅克  

卡斯帕：死人不会再痛。  
杰洛特：你问过他们吗？  
莉达：你问过母亲愿不愿意埋孩子吗？  
杰洛特：你们用别人的死拖住自己的活。  
莉达：活着的人都这么做，只是我们被你抓到了。

### 场景 2.4：选择

**选择 A：公开真相**  
**写入：** `bone_rib = geralt`，`public_truth += 1`，`village_fear += 1`，`melowa_truth += 1`  
**后果脚本：** 村民聚集，卡斯帕一家被驱逐。若 `village_fear >= 2`，暴民会试图烧草药师小屋。

**选择 B：私下解除诅咒**  
**写入：** `bone_rib = buried`，`village_fear -= 1`，`melowa_truth += 1`  
**后果脚本：** 亚涅克高烧，伊蕾娜留下治疗。村民不知道完整真相。

**选择 C：让夫妇带孩子逃走**  
**写入：** `bone_rib = family_lost`，`public_truth -= 1`，`melowa_truth += 1`  
**后果脚本：** 后续长线缺少肋骨证据，但可通过特莉丝信件补足部分推理。

## 任务 3：《红蜡印》

### 任务日志

圣卡莱门修道院声称圣物失窃，嫌疑人是一个草药师。门锁没有撬痕，圣匣上的红蜡却被熟人打开。神迹也许被偷了，也许只是被卖了。

### 场景 3.1：修道院委托

**地点：** 圣卡莱门修道院  
**参与者：** 杰洛特、院长维克托  

维克托：圣物被亵渎。若不找回，信众会动摇。  
杰洛特：你担心圣物，还是信众？  
维克托：二者有区别吗？  
杰洛特：对圣物来说没有。对信众来说有。

### 场景 3.2：地窖灰烬

**地点：** 修道院地窖  
**参与者：** 杰洛特、特莉丝  
**触发条件：** 玩家发现烧毁记录并选择联系特莉丝。  

特莉丝：这地方到处都是猎巫人的眼线。  
杰洛特：所以你穿了灰斗篷。没人会注意一个穿灰斗篷的红发女术士。  
特莉丝：继续说，我就让你自己读灰。  
杰洛特：我认识的字还没烧完。  
特莉丝：烧完的才重要。

**恢复文本：**  
“梅洛娃，民间医者，未经神殿许可行奇迹。拒绝交出病患名单。收容非人、孤儿与隔离者。建议死后封圣，以纠正其生前异端倾向。”

### 场景 3.3：处决前对峙

**地点：** 修道院广场  
**参与者：** 杰洛特、维克托、诺瓦克、芙蕾娜、群众  

诺瓦克：草药师偷圣骨，异端偷神迹。故事简单得很。  
杰洛特：简单故事通常少了一个拿钱的人。  
维克托：你在圣地污蔑神仆。  
杰洛特：你的戒指上有红蜡。猎巫人的马袋里有掌骨。你们谁想先解释？

### 场景 3.4：选择

**选择 A：公开揭露院长**  
**写入：** `bone_palm_church = geralt`，`public_truth += 1`，`witch_hunter_attention += 1`，`melowa_truth += 1`  
**后果：** 修道院声望下降；诺维格瑞猎巫巡逻增加。

**选择 B：救走芙蕾娜并销毁证据**  
**写入：** `bone_palm_church = hidden`，`witch_hunter_attention -= 1`，`melowa_truth += 1`  
**后果：** 芙蕾娜存活；教团保住表面权威。

**选择 C：把证据交给地下情报商**  
**写入：** `bone_palm_church = broker`，`blackmail_evidence = true`，`melowa_truth += 1`  
**后果：** 最终任务可用黑账压制教团，但地下势力会索取回报。

## 任务 4：《没有脸的施主》

### 任务日志

盐沟贫民区的匿名施主救了很多人。现在受救济者在夜里梦游自残，嘴里念着陌生人的名字。慈善有时候不是没有代价，只是账单寄给了别人。

### 场景 4.1：梦游者

**地点：** 盐沟巷道  
**玩法：** 非致命制止梦游者。  
**失败条件：** 玩家用致命攻击会导致 `public_truth -= 1`，并关闭部分证词。  

梦游者：不是我的手。不是我的病。别关门，别关门。  
杰洛特：他们不是被控制。是在重复记忆。

### 场景 4.2：丹德里恩认出禁歌

**地点：** 盐沟酒馆  
**参与者：** 杰洛特、丹德里恩、酒客  

丹德里恩：停。刚才那句再唱一遍。  
酒客：你付钱吗？  
丹德里恩：我付给艺术，不付给敲诈。  
杰洛特：他会付。  
丹德里恩：感谢你一如既往地支持我的破产。

**禁歌歌词：**  
她藏起尖耳的孩子，  
她把门闩插上，  
她说神若进屋，  
也要先洗手。

### 场景 4.3：艾勒的真身

**地点：** 施粥点后屋  
**参与者：** 杰洛特、艾勒、修女玛塔  

艾勒：如果我停下，他们会来清洗。  
杰洛特：他们知道你是什么？  
艾勒：知道。只要锅还冒热气，他们愿意假装不知道。  
玛塔：这些人会死。没有圣骨，他们撑不过冬天。  
杰洛特：有圣骨，他们梦里替别人死。

### 场景 4.4：选择

**选择 A：揭露艾勒**  
**写入：** `bone_palm_soup = geralt`，`witch_hunter_attention += 1`，`melowa_truth += 1`  
**后果：** 艾勒逃亡或死亡；盐沟清洗风险上升。

**选择 B：保护艾勒并毁掉圣骨环**  
**写入：** `bone_palm_soup = destroyed`，`witch_hunter_attention -= 1`，`public_truth += 1`，`melowa_truth += 1`  
**后果：** 救济点失去奇迹，但教团失去借口。

**选择 C：逼艾勒交出黑账**  
**写入：** `bone_palm_soup = geralt`，`blackmail_evidence = true`，`witch_hunter_attention += 1`，`melowa_truth += 1`  
**后果：** 最终可压制教团；艾勒可能遭报复。

## 任务 5：《潮下钟声》

### 任务日志

洛恩维克岛没有钟，海底却夜夜传来钟声。被钟声引走的船员说他们听见死去亲人的呼救。岛民说那是海妖。叶奈法说，别急着相信岛民。

### 场景 5.1：海雾与空钟架

**地点：** 洛恩维克岸边  
**参与者：** 杰洛特、凯拉赫、哈尔格  

凯拉赫：我丈夫听见女儿叫他。她死了六年。  
哈尔格：海会拿走想听见死人说话的人。  
杰洛特：也会拿走说谎的人。  
哈尔格：那这岛早该沉了。

### 场景 5.2：叶奈法读取咒式

**地点：** 海底旧钟  
**玩法：** 潜水调查，躲避溺水妖灵和海藤。  
**参与者：** 杰洛特、叶奈法  

叶奈法：有趣。  
杰洛特：你说有趣的时候，通常有人要倒霉。  
叶奈法：已经倒霉了。这个咒式不是封印怪物，是封印一种权柄。  
杰洛特：指定谁死的权柄？  
叶奈法：更准确地说，指定谁替别人活下去付账。

### 场景 5.3：氏族旧誓

**地点：** 氏族厅  
**参与者：** 杰洛特、叶奈法、哈尔格、凯拉赫  

哈尔格：她求我们把骨头沉下去。说如果有人把她合回去，海以外的地方也会听见钟。  
凯拉赫：你知道钟声为什么杀人，却让我们把死人归给海妖？  
哈尔格：誓言不是给活人舒服用的。  
杰洛特：也不是给活人撒谎用的。

### 场景 5.4：选择

**选择 A：维持氏族秘密**  
**写入：** `bone_leg = sea`，`clan_oath += 0`，`melowa_truth += 1`  
**后果：** 岛上稳定；最终缺少公开见证。

**选择 B：公开梅洛娃遗愿**  
**写入：** `bone_leg = witnessed`，`clan_oath += 1`，`public_truth += 1`，`melowa_truth += 1`  
**后果：** 氏族在最终任务中护送杰洛特；哈尔格名誉受损。

**选择 C：带走腿骨**  
**写入：** `bone_leg = geralt`，`clan_oath -= 1`，`melowa_truth += 1`  
**后果：** 可强行重组五骨；史凯利格追兵在最终任务出现。

## 长线任务：《五骨归龛》

### 长线触发规则

| 条件 | 触发内容 |
|---|---|
| `melowa_truth >= 2` | 触发第一幕“圣女传说”。 |
| `melowa_truth >= 3` | 触发第二幕“医者真相”。 |
| 完成《潮下钟声》或 `melowa_truth >= 4` | 触发第三幕“禁忌代价”。 |
| 取得丹德里恩歌谣证据或 `blackmail_evidence = true` | 触发第四幕“被制造的圣女”。 |
| `melowa_truth >= 4` 且至少完成四个短线任务 | 触发第五幕“最后见证”。 |

### 第一幕：圣女传说

**地点：** 旅店房间 / 可复用中立室内场景  
**参与者：** 杰洛特  
**玩法：** 线索板整理。  

**杰洛特自语：**  
灰垄的白手，白水轮的药方，修道院的圣匣。三个地方，同一个名字。不是巧合。也不是圣迹。

**目标更新：** “继续调查圣梅洛娃遗骨的来源。”

### 第二幕：医者真相

**地点：** 诺维格瑞安全屋或密信界面  
**参与者：** 杰洛特、特莉丝  

特莉丝：她不是学院术士。也不是神殿医师。  
杰洛特：民间医者。  
特莉丝：对。聪明、绝望、没有保护。最容易被利用的那种人。  
杰洛特：她的药方救过人。  
特莉丝：也害过人。两件事可以同时为真。

### 第三幕：禁忌代价

**地点：** 史凯利格海岸或叶奈法临时实验桌  
**参与者：** 杰洛特、叶奈法  

叶奈法：完整圣骨能恢复原始仪式。  
杰洛特：治愈瘟疫？  
叶奈法：转嫁瘟疫。  
杰洛特：区别在尸体是谁。  
叶奈法：你今天格外敏锐。

### 第四幕：被制造的圣女

**地点：** 诺维格瑞酒馆  
**参与者：** 杰洛特、丹德里恩  

丹德里恩：我找到了最早的版本。没有圣女，没有金光，没有跪满地的信徒。只有一个女人、一间破屋和一群没人愿意碰的病人。  
杰洛特：后来呢？  
丹德里恩：后来人们决定，一个死人最好用。她不会纠正歌词。

**若 `blackmail_evidence = true`：**

丹德里恩：还有这个。账册。修道院、猎巫人、商人。你知道什么比圣歌更容易传播吗？  
杰洛特：丑闻。  
丹德里恩：终于，你开始理解艺术了。

### 第五幕：最后见证

**地点：** 旧战线废弃驿站  
**参与者：** 杰洛特、希里、叶奈法、特莉丝、丹德里恩、派系代表  
**触发：** 玩家选择进入最终汇聚任务。  

希里：我不是来替你选。  
杰洛特：我知道。  
希里：我能听见她。不是圣女。一个很累的人。  
杰洛特：她说什么？  
希里：她说，不要让他们再把我合回去。

**记忆显现对白：**

梅洛娃：我救过人。也害过人。我以为只要多给他们一个冬天，他们会找到更好的办法。可掌权者会把暂时的代价写成律法。不要让任何人再用我的骨头决定谁替别人死。

## 最终选择脚本

### 选择 A：重组圣骨，执行仪式

**可用条件：** 至少取得四块圣骨控制权。  
**玩家选项文本：** “执行仪式。让多数人活下来。”  

**杰洛特：**  
我不是来找干净答案的。没有。只有活下来的人和付账的人。

**结局旁白：**  
瘟疫没有进城。麦田没有全烂。神殿说圣女归来，村长们说白狼终于明白了牺牲。没人问被选中的人怎么想。死人很少被邀请参加庆典。

### 选择 B：摧毁全部遗骨

**可用条件：** `melowa_truth >= 4`，叶奈法完成反咒。  
**玩家选项文本：** “摧毁圣骨。结束转嫁。”  

**叶奈法：** 这会释放被压住的东西。  
**杰洛特：** 总得有人停止把它推给别人。  
**叶奈法：** 很高尚。也会很血腥。  
**杰洛特：** 两件事可以同时为真。

**结局旁白：**  
圣骨烧成灰后，瘟疫回到了它本该去的地方：所有地方。人们说白狼毁了奇迹。也有人说，至少这一次，死亡没有被偷偷分配给最容易被遗忘的人。

### 选择 C：公开真相并分散遗骨

**可用条件：** `public_truth >= 2`，且拥有丹德里恩旧歌或 `blackmail_evidence = true`。  
**玩家选项文本：** “公开真相。分散遗骨。”  

**杰洛特：**  
她不是圣女。也不是怪物。她是人。你们把她用成了两者。

**丹德里恩：**  
我可以唱。可他们未必会听。

**杰洛特：**  
那就唱到他们没法说没听见。

**结局旁白：**  
教团失去完整圣迹。村庄失去简单答案。史凯利格重新立誓。诺维格瑞的酒馆里有人唱起没有圣字的新歌。瘟疫仍来，饥荒仍来，权力仍找新的壳。但梅洛娃的骨头不再合拢，她的名字也不再只属于神殿。

### 选择 D：交给某一方

**可用条件：** 至少一个派系在最终场景中保持可交易状态。  
**玩家选项文本：** “把圣骨交给他们。换取局部稳定。”  

**派系代表通用对白：**  
你会得到报酬，猎魔人。还有秩序。人们需要秩序，胜过他们需要真相。

**杰洛特：**  
人们常这么说。通常是替别人决定的时候。

**结局旁白：**  
圣骨有了新的主人。圣女有了新的故事。杰洛特离开时，已经有人开始争论梅洛娃究竟想要什么。死人沉默，活人替她说话。生意照旧。

## 失败与缺证分支

### 梅洛娃真相不足

**条件：** `melowa_truth < 4` 进入最终。  
**效果：** 希里记忆场景残缺，结局 B 和 C 不可用或需要更高代价。  

希里：我只听见碎片。海水、火、孩子哭声。还有她说“分开”。但不够。  
杰洛特：不够说服他们。  
希里：也不够说服你自己。

### 猎巫压力过高

**条件：** `witch_hunter_attention >= 2`  
**效果：** 特莉丝不在最终现场，只通过密信提供证据；艾勒或芙蕾娜可能死亡。  

密信：  
我不能来。街上全是猎巫人。别把这当成怯懦，杰洛特。活着的人有时比死去的证据更难保存。

### 村社恐惧过高

**条件：** `village_fear >= 2`  
**效果：** 威伦村民拒绝公开作证，公开真相结局需要 `blackmail_evidence = true` 才可用。  

威伦村民：你拿走骨头，拿走粮，现在还要拿走我们最后能信的东西？

### 氏族誓约破裂

**条件：** `clan_oath < 0`  
**效果：** 最终场景出现史凯利格追兵；结局 C 需要战斗或谈判额外节点。  

哈尔格：你带走了她要求沉海的骨头。别再用遗愿这个词。

## 制作备注

- 五个短线任务应允许乱序完成，但每个任务必须至少提供一条 `melowa_truth` 核心证据。
- 正史角色不能替玩家做最终选择，只能改变证据强度、风险和可用结局。
- 所有“好结果”都要保留代价：疾病、饥荒、暴力、信仰崩塌或政治报复。
- 梅洛娃不能被写成纯粹无辜圣人。她救人，也犯错；真正的冲突来自后人把她的错误制度化。
- 最终任务不能变成收集五件神器后的奖励仪式。它必须让玩家意识到：完整圣骨本身就是危险。
