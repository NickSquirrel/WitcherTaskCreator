# 输出模板

当用户需要结构化交付物时使用这些模板。根据请求规模删减或扩展标题。

## 小型独立任务

```markdown
## 任务标题

**类型：** 猎魔委托 / 调查任务 / 政治支线 / 诅咒案件
**推荐等级或规模：**
**地点：**
**语气：**

### 一句话前提

### 委托钩子

### 主要角色

### 表面问题

### 隐藏真相

### 调查流程

1.
2.
3.

### 关键线索

### 选择点

### 结局

### 奖励

### 后续回响
```
